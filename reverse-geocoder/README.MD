reverse_geocoder
===
A fast, offline reverse geocoder.

## Documentation

https://docs.rs/reverse_geocoder/

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
reverse_geocoder = "3.0"
```

## Benchmark

`cargo bench` on M1:

```
> cargo bench
search                  time:   [489.30 us 490.03 us 490.94 us]
```
